Configuring the Media Library

The Media Library displays all assets uploaded in the Strapi application. The Media Library settings allow controlling the format, file size, and orientation of uploaded assets. Those settings can be configured from Settings > Global settings > Media Library.

To configure the Media Library settings:

  1. Go to the Global settings > Media Library sub-section of the settings interface.

  2. Define your chosen new settings:

    Setting nameInstructionsDefault value
    Responsive friendly uploadEnabling this option will generate multiple formats (small, medium and large) of the uploaded asset.True
    Size optimizationEnabling this option will reduce the image size and slightly reduce its quality.True
    Auto orientationEnabling this option will automatically rotate the image according to EXIF orientation tag.False

  3. Click on the Save button.
